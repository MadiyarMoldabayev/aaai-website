# AAAI Website

Современный веб-сайт Ассоциации по развитию искусственного интеллекта (AAAI).

## Особенности

- Современный, отзывчивый дизайн
- Секции: О нас, Членство, Филиалы, Комитеты, События, Ресурсы
- Плавная прокрутка и анимации
- Мобильная навигация
- Оптимизирован для производительности

## Структура

- `index.html` - Главная страница
- `css/style.css` - Стили
- `js/main.js` - JavaScript функциональность

## Развертывание

Сайт развернут на Netlify и автоматически обновляется при push в GitHub.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- Netlify для хостинга
